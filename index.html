DELL R730 H730P RAID UBUNTU 14.04.3 install

1.系统安装

ubuntu 14.04.3 server amd 64

2.hardware RAID

开机ctrl+R 配置 raid1

系统安装时可见 H730P mini 2T

硬raid管理工具:LSIMegaRAIDSAS
详细:HWraid

由于没有提供14.04的源,12.04凑合用吧

在/etc/apt/source.list中添加deb http://hwraid.le-vert.net/ubuntu precise main


执行wget -O - http://hwraid.le-vert.net/debian/hwraid.le-vert.net.gpg.key | sudo apt-key add -和sudo apt-get update


安装sudo apt-get install megaraid-status后可执行megaraidsas-status和sudo megasasctl查看raid状态。

3.install gnome desktop for XXX

sudo apt-get update; sudo apt-get install gnome-session-fallback
